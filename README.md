# Stacknity Technologies

Welcome to the **Stacknity Technologies** corporate website repository.

Stacknity Group is focused on building digital empires through scalable technology solutions, enterprise-grade architecture, and elite global engineering talent deployment. From high-performance web platforms and mobile apps to comprehensive staff augmentation, our goal is to deliver excellence and measurable ROI.

## 🌟 Features

*   **Modern Web Architecture**: Single Page Application structure engineered for ultra-fast, seamless loading and transitions.
*   **Immersive 3D Experiences**: Integrated WebGL experiences powered by Three.js, including interactive background particles, geometry nodes, and a premium holographic tech sphere hero element.
*   **Premium Design Aesthetic**: Uses a rich dark-mode theme featuring cyan (`#00E5FF`) and deep violet (`#8A2BE2`) gradients, immersive glowing effects, and a deep-blur glassmorphism ui.
*   **Responsive Flow**: Fully optimized fluid layouts that scale perfectly across desktop, tablet, and mobile devices natively.
*   **Micro-Interactions**: Features a custom cursor, scroll-triggered reveals, continuous tech stack marquees, and dynamic hover-reactive elements to maximize engagement.

## 🚀 Getting Started

To run the project locally, simply clone the repository and serve the static files:

1. Clone the repo:
   ```bash
   git clone https://github.com/Stacknity-Group/stacknity-technologies.git
   ```
2. Navigate into the directory:
   ```bash
   cd stacknity-technologies
   ```
3. Use your preferred local development server (e.g., Live Server, Python HTTP server, or Node `http-server`):
   ```bash
   npx http-server . -p 8888
   ```
4. Open your browser and navigate to `http://localhost:8888`.
